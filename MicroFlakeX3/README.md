# MicroFlakeX3  
* MicroFlakeX3包含:  
> MfxBase：MFX基类。支持反射、工厂模式、线程安全  
> MfxType：MFX基本类型(包含了Rect、Size、Point、Color等解基本类型)  
> MfxGraph_GDIP：MFX基本绘制，基于GDI+(包含MfxCanvas-MfxImage-MfxWorlds-MfxShape)  
> MfxGraph_D2D1：MFX基本绘制，基于D2D(包含MfxCanvas-MfxImage-MfxWorlds-MfxShape)  
> MfxAppFrame：MFX程序框架  
# MicroFlakeX3简介  
* MFX3的目标是让WinAPP开发远离复杂的框架，专注于开发本身。  
* MFX3可以快速自定义控件、窗口的背景图片、蒙版，让你的程序变得更加好看  
* MFX3非常简单，正是因为它的简单，所以容易上手并且高度可自定义。
